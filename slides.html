Hardware: What's electricity?

  • Electrons jumping from one atom to the next
  • Volts: How many electrons are moving?
  • Amps: Power! How fast are they going?
  • Closed circuits, directionality matters
  • Well actually: voltage measures the difference in charge between two points

Safety

Don't let the smoke out

  • Always kill the power before modifying a circuit
  • Some parts (LEDs, servos, sensors) care about direction; mind the GND
  • Do not get wet; water is a conductor
  • High amperage is what will kill you; unlikely to encounter
  • Always wear safety glasses if you're cutting wire
  • Solder is a burn risk; also wear glasses
  • Avoid soldering and wire crimping with a breadboard
  • Convention: power rails, red = +5v / black = GND, standard form-factor for ICs and components

Breadboard

A rapid prototyping framework
  • Measures inputs, provides output and makes decisions
  • Power and communication from USB

CIRC-01-code-led-a-strobe.js:

var j5 = require("johnny-five");
var myBoard, myLed;
myBoard = new j5.Board();
myBoard.on("ready", function() {
  myLed = new j5.Led(13);
  myLed.strobe( 1000 );
  this.repl.inject({ // make myLED available as "led" in REPL
      led: myLed
  });
  // try "on", "off", "toggle", "strobe", "stop"
});

Demo: Motor

http://node-ardx.org/exercises/3:

> node code/CIRC03-code-motor.js

CIRC03-code-motor.js:

var five = require("johnny-five"), board, myMotor, led;
board = new five.Board();
board.on("ready", function() {
  myMotor = new five.Motor({
    pin: 9
  });
  myMotor.on("start", function( err, timestamp ) { // event handlers on start and stop
    console.log( "started", timestamp );
    board.wait(2000, function() { // stop after 2 seconds
      myMotor.stop();
    });
  });
  myMotor.on("stop", function( err, timestamp ) {
    console.log( "stopped", timestamp );
  });
  myMotor.start();
});
